# Microservicio Abastecimiento

Gestiona proveedores, ordenes de compra y recepcion de mercaderia.

## Endpoints principales

- `GET /api/proveedores`
- `POST /api/proveedores`
- `PATCH /api/proveedores/{id}/estado`
- `GET /api/ordenes-compra`
- `POST /api/ordenes-compra`
- `PATCH /api/ordenes-compra/{id}/aprobacion`
- `PATCH /api/ordenes-compra/{id}/cancelacion`
- `POST /api/recepciones`
- `PATCH /api/recepciones/{id}/confirmacion`
